## install
```
sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
```

## remove
```
sudo apt-get remove atom
```

## packages
### general
* sync-settings 同步atom设置
* vim-mode-plus
* markdown-preview
* terminal plus

### git
* git-plus
* git-project

### coding
* docblockr 方便写注释
* autocomplete-clang 代码补全
* linter-clang 语法检测
* clang-format 格式化代码

### convenience
* autocomplete-paths
