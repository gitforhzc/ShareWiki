## install
```
sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
```

## remove
```
sudo apt-get remove atom
```

## packages
### general
* sync-settings 同步atom设置
* vim-mode
* markdown-preview

### git
* git-plus
* git-control
* git-project

### code
* docblockr 方便写注释
* linter-gcc 代码校验工具
* atom-beautify 格式化代码
